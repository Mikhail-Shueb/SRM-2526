# Step 5: CLIK

The final report text for this step is in:

- `step5_clik_report.md`

The additional numerical experiments used in the report are generated by:

- `step5_clik_experiments.m`

## How to Run

```matlab
cd('SRM/project')
run('generate_jacobian_library.m')
run('generate_clik_model.m')
run('validate_clik.m')
run('step5_clik_experiments.m')
```

Once you run everything you can open the simulink model:

```matlab
open_system('Kuka_CLIK_Model')
```
